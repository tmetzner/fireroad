# Brand Assets & Style Guide

Upload your brand assets and style guide files here. These will be used as reference when developing the website.

## Folder Structure

- **logos/** — Logo files (SVG, PNG, etc.) in various formats and sizes
- **colors/** — Color palettes, swatches, or reference files
- **typography/** — Font files (.woff, .woff2, .ttf, .otf) and type specimens
- **icons/** — Icon sets and individual icon files
- **images/** — Hero images, backgrounds, photography, and other visual assets

## Style Guide

Place your style guide document (PDF, image, or any format) directly in this folder.

## Supported File Types

You can upload any file type here — PDFs, images, font files, Figma exports, Sketch files, etc.
